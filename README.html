Polybar

For building polybar your will first need to install yay.

cd ~
mkdir -p /tmp/yay_install
cd /tmp/yay_install
sudo pacman -S base-devel
sudo pacman -S expac yajl git
git clone https://aur.archlinux.org/yay.git 
cd yay
makepkg -si 
cd ~ 
rm -rf /tmp/yay_install
After that you can build polybar by 
sudo pacman -S cairo libxcb python2 xcb-proto xcb-util-image xcb-util-wm xcb-util-xrm jsoncpp
yay -S polybar-git
Setting up the wallpaper Create a Pictures Directory and inside of that create a Wal directory.
cd ~
mkdir Pictures
cd Pictures
mkdir Wal
now paste the wallpaper from Resources into this directory.
